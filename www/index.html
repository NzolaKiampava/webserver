Servidor HTTP implementado em C++ 98

✓ Servidor Online

Bem-vindo ao Webserv! Este é um servidor web funcional desenvolvido do zero em C++ 98 para o projeto 42.

Principais Características

  • Recebe requisições HTTP de navegadores reais
  • Processamento e resposta adequada a requisições
  • Serviço de websites estáticos completos
  • Execução de scripts CGI (PHP, Python, etc.)
  • Gerenciamento de múltiplas conexões simultaneamente
  • Sem bloqueio de I/O (Non-blocking)
ℹ️ Informação: Este servidor está funcionando perfeitamente. Você pode começar a fazer requisições!
